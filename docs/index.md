# Sortify — Documentação

Bem-vindo à documentação do **Sortify**, organizador de arquivos para Windows com GUI e CLI.

Nesta documentação você encontra:

- visão geral da ferramenta;
- instalação local;
- configuração por `config.json`;
- publicação da documentação com **MkDocs** no **GitHub Pages**.

## Sobre o projeto

O Sortify automatiza a organização de arquivos em pastas como Downloads e Desktop, com suporte a regras personalizadas, detecção de duplicados, modo simulação e desfazer.

> Repositório principal: `desktop-helper`
