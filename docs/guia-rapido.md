# Guia rápido

## Pré-requisitos

- Python 3.10+
- `pip`

## Instalação

```bash
git clone https://github.com/seu-usuario/desktop-helper.git
cd desktop-helper
pip install -r requirements.txt
```

## Executando

### GUI

```bash
python organizador_gui.py
```

### CLI

```bash
python organizador_cli.py
```
